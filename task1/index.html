HTML Forms

Svi mi , uglavnom, znamo šta je to forma i kako izgleda popunjavanje forme (što bismo rekli "po naški" - formular). A kako izgleda HTML form-a i kako je napraviti, možete otkriti ukoliko pritisnete na dugme ispod.

Hajde da otkrijemo

< form > Element

HTML < form > element se koristi za pravljenje HTML formulara i prikupljanje unešenih podataka od strane korisnika. Gde se prikupljene informacije, u pozadini, šalju dalje na obradu.

Svaka forma sadrži neke od sledećih elemenata:

  • <label>
  • <input>
  • <textarea>
  • <select>
  • <datalist>
  • <option>

<input> element je najkorišćeniji element forme i može biti prikazan na nekoliko različitih načina, u zavisnosti od type atributa:

Vrste <input> elementa
Type Objašnjenje
<input type="text"> Prikazuje polje za unos teksta
<input type="password"> Prikazuje polje za unos lozinke
<input type="number"> Prikazuje polje za unos broja
<input type="range"> Prikazuje klizača
<input type="radio"> Prikazuje radio dugme (za selekciju jednog od više izbora)
<input type="checkbox"> Prikazuje kvadrat za čekiranje (više izbora ili ni jednog)
<input type="submit"> Prikazuje dugme za slanje (podnošenje formulara)
<input type="button"> Prikazuje dugme na koje se može kliknuti

HTML <form> atributi
Vrsta Objašnjenje
<form action="/"> Action atribut definiše radnju koja će se izvršiti kada se obrazac pošalje.
<form target="_blank"> Target atribut određuje gde da se prikaže odgovor koji je primljen nakon slanja obrazca
<form method="POST"> Method atribut određuje HTTP metod koji će se koristiti prilikom slanja podataka obrazca.
<form autocomplete="on"> Određuje da li forma treba da ima uključeno ili isključeno automatsko popunjavanje
